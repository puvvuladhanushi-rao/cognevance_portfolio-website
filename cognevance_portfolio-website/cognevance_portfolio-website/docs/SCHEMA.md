# Database Schema — Portfolio Contact Form

This project uses a single collection since the only persisted data is contact
form submissions. If you extend the project later (e.g. a blog or project
CMS), add new collections/tables following the same pattern.

## MongoDB (Mongoose) — `messages` collection

| Field      | Type     | Required | Notes                                  |
|------------|----------|----------|-----------------------------------------|
| `_id`      | ObjectId | auto     | Generated by MongoDB                    |
| `name`     | String   | yes      | Max 100 chars, trimmed                  |
| `email`    | String   | yes      | Validated against a basic email regex   |
| `subject`  | String   | no       | Max 150 chars, defaults to "No subject" |
| `message`  | String   | yes      | Max 2000 chars                          |
| `isRead`   | Boolean  | no       | Defaults to `false`, for admin triage   |
| `createdAt`| Date     | auto     | Added by `timestamps: true`             |
| `updatedAt`| Date     | auto     | Added by `timestamps: true`             |

Example document:

```json
{
  "_id": "665f1c2e8a1b2c3d4e5f6789",
  "name": "Jane Doe",
  "email": "jane@example.com",
  "subject": "Loved your portfolio",
  "message": "Would love to connect about a freelance opportunity.",
  "isRead": false,
  "createdAt": "2026-07-06T10:15:00.000Z",
  "updatedAt": "2026-07-06T10:15:00.000Z"
}
```

## If you'd rather use PostgreSQL / SQL

```sql
CREATE TABLE messages (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  email VARCHAR(255) NOT NULL,
  subject VARCHAR(150) DEFAULT 'No subject',
  message TEXT NOT NULL,
  is_read BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

## Design notes

- Field length limits guard against abuse of the public-facing form.
- `isRead` gives you a simple way to build an admin inbox view later without
  a schema migration.
- Timestamps are handled automatically rather than set manually, so they
  can't be spoofed by the client.
